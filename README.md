# Spring-Hibernate-Student-Management

A mini project demonstrating:
- Spring DI with Java-based configuration
- CRUD operations using Hibernate ORM
- Transaction Management for fee payment and refund
- Integration of Spring + Hibernate for real-world data management

## Modules
- Student management (CRUD)
- Fee/Refund transaction module

## How to run
- Standard Spring Boot app with in-memory H2 for demo
- See src/main/resources/application.properties
